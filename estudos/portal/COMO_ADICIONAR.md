# Como Adicionar Conteúdo ao Portal

**Arquivo principal:** `index.html` (mesma pasta)
**Documentação completa:** `../../projects/portal-estudos.md`

## Fluxo rápido

1. Me envie o conteúdo da matéria/prova
2. Eu leio `portal-estudos.md` para pegar os templates (custo mínimo)
3. Faço Edit cirúrgico no `index.html`
4. Pronto — sem reconstruir o site inteiro

## O que posso adicionar

- Nova matéria completa (teoria + exercícios + flashcards + vídeos)
- Exercícios avulsos para matéria existente
- Novo simulado de vestibular (qualquer ano/banca)
- Vídeos para matéria existente
- Diagrama visual para tópico de teoria

## Tipos de conteúdo aceitos

- Foto/texto do caderno ou apostila
- PDF de prova anterior
- Lista de exercícios
- Enunciado de questões de vestibular
- Qualquer conteúdo escolar
