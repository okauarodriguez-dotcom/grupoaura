# Skill: /standup

Gere um briefing executivo rápido para começar o dia ou uma reunião:
1. Leia projects/ — liste lançamentos e projetos ativos com status atual
2. Leia campanhas/ — mostre campanhas rodando e últimas métricas/observações registradas
3. Leia decisoes/ — mostre as 3 decisões mais recentes e se há pendências
4. Leia people/ — verifique se há follow-ups ou ações pendentes com parceiros/time
5. Mostre os próximos passos mais críticos em ordem de prioridade

Seja direto e objetivo — sem floreio.
