# Skill: /capturar

Regras de captura automática para o vault Obsidian do Kauã.

## Pessoa nova mencionada
Criar imediatamente `people/nome-sobrenome.md` com papel, empresa e contexto. Não perguntar, só criar.

## Livro novo mencionado
Criar imediatamente `livros/titulo-do-livro.md` com resumo completo + conexão com o negócio de DR/infoprodutos/nutraceuticos do Kauã.

## Projeto novo mencionado
Criar imediatamente `projects/nome-do-projeto.md` com objetivo, status e contexto.

## Reunião nova
Criar imediatamente `reunioes/YYYY-MM-DD-contexto.md` com resumo, participantes e próximos passos. Usar /reunião se houver transcrição.

## Decisão importante
Criar imediatamente `decisoes/YYYY-MM-DD-contexto.md` com a decisão e o raciocínio.

## Context Rules
- Decisão mencionada → checar decisoes/ primeiro
- Pessoa mencionada → buscar em people/, criar se não existir
- Produto ou campanha → buscar em produtos/ ou campanhas/
- Projeto mencionado → buscar em projects/, criar se não existir
- Inbox/ com novo item → perguntar se quer organizar agora
- Pedido de escrita → ler notas recentes em daily/ para capturar tom de voz
- Livro mencionado → buscar em livros/, criar se não existir
- Mentoria ou aula → buscar em mentoria/
