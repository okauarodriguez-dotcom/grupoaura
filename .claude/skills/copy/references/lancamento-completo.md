## FASE 1 — Captação de Leads (D-14 a D-8)

### Anúncios de CPL (Meta / TikTok)
**Objetivo:** Trazer prospects qualificados para o funil (email + WhatsApp + grupo).

**Copy do Anúncio — Primary Text (tráfego frio):**
```
[Avatar qualificado + dor em 1 linha]

[MUP em 1 linha — por que o problema existe]

[O que vão aprender — benefício do CPL, não do produto]

[Link] → [CTA com resultado, não "saiba mais"]
```

**Exemplo para lançamento de infoproduto:**
```
Se você [tem problema X] e já tentou [solução Y] sem resultado...

Não é falta de esforço. É porque ninguém te ensinou [MUP].

Vai ter uma aula gratuita onde vou mostrar [resultado específico que a aula entrega].

[Link] → Quero garantir minha vaga
```

**Headline do ad:** "[Resultado específico] — Aula gratuita com [autoridade]"
**Descrição:** "Vagas limitadas. Sem cobranças."

---

### Posts de Feed (Aquecimento D-14 a D-8)
3-4 posts de valor que estabelecem o problema e o vilão antes de anunciar qualquer coisa:

**Post 1 — Problema:**
```
[Dado surpreendente sobre o problema do avatar]

[Por que acontece — MUP em linguagem acessível]

[Qual a consequência invisível de não resolver]

Amanhã posto mais sobre isso. 👇
```

**Post 2 — Vilão:**
```
A culpa não é sua se você [tentou e falhou].

O sistema [indústria/crença comum] foi construído para te manter em [problema].

Aqui está o motivo: [explicação do vilão em 3-4 linhas]

[Pergunta de engajamento relacionada à dor]
```

**Post 3 — Teaser:**
```
Semana que vem vou mostrar o que [número] pessoas usaram para [resultado específico].

É de graça. Precisa só reservar [tempo da aula].

Comenta "QUERO" aqui embaixo que te mando o link quando abrir.
```

## FASE 2 — Aulas de Conteúdo (D-7, D-5, D-3)

### Estrutura Geral de Cada Aula

**Abertura (3-5 min) — Hook + Qualificação**
```
"Antes de começar, quero saber: quantos aqui já [tentativa falhada]? Coloca no chat."
[Espera respostas]
"Exato. E se você tá aqui é porque sabe que existe uma forma melhor."
"Hoje vou te mostrar [promessa específica da aula — não do produto]."
```

**Corpo (20-40 min) — Conteúdo Real + Construção de Crença**
- Entrega valor genuíno antes de qualquer pitch
- Apresenta o vilão e o MUP em contexto de educação
- Usa depoimentos como ilustração de ponto educativo, não como prova de produto
- Constrói a crença que o produto resolve antes de mencionar o produto

**Fechamento de Aula (5-10 min) — Teaser e Comprometimento**
```
"Na próxima aula vou mostrar exatamente [o que a próxima aula entrega]."
"Mas antes: quem aqui conseguiu [resultado da aula]? Conta no chat."
[Engaja com respostas]
"Importante: na [data], vou abrir acesso a algo que vai te ajudar a [resultado do produto]. Fica de olho."
```

---

### Aula 1 (D-7) — O Problema Real
**Objetivo:** Nomear o vilão. Absolver o avatar. Criar curiosidade sobre a solução.
- Conteúdo: MUP explicado em profundidade
- Crença a construir: "O problema não é minha culpa — tem uma causa específica"
- Teaser: "Na próxima aula vou mostrar o que muda quando você resolve a causa raiz"

### Aula 2 (D-5) — O Mecanismo
**Objetivo:** Revelar o MUS. Mostrar que existe solução. Criar desejo.
- Conteúdo: Como o mecanismo funciona + primeiros resultados possíveis
- Crença a construir: "Existe uma forma diferente que eu ainda não tentei — e ela faz sentido"
- Teaser: "Na terceira aula vou te mostrar como aplicar isso de forma completa — e vou revelar algo especial"

### Aula 3 (D-3) — A Solução + Abertura de Carrinho
**Objetivo:** Completar a jornada educativa + fazer pitch natural do produto.
- Conteúdo: Aplicação completa do mecanismo + casos reais
- Crença a construir: "Preciso disso e faz sentido investir"
- Pitch no final: "Para quem quer aplicar isso com [estrutura/suporte/material completo], estou abrindo acesso a [produto]. Mas só a partir de [data]."

## FASE 4 — Carrinho Aberto (D0 a DC)

### D0 — Manhã — Abertura Oficial

**Email de Abertura:**
**Assunto:** "[Produto] abriu — aqui está o acesso"
```
[Nome],

É hoje.

[Nome do produto] está disponível.

[Bloco de oferta completo:]
Você leva:
• [Produto core + resultado]
• [Bônus 1 — nome + resultado + valor]
• [Bônus 2 — nome + resultado + valor]
• [Bônus 3 — nome + resultado + valor]

Total: R$[X] de valor

Por R$[preço] — [razão justificada]

Garantia de [N] dias: [framing como argumento]

[Link] → [CTA com identidade]

P.S. Essa condição está disponível até [data/hora] por [razão real].
```

**VSL de Abertura de Carrinho (se houver):**
Usar estrutura de VSL completo (14 blocos). Duração ideal: 20-35 min.
- Lead: história ou revelação de mecanismo (dependendo do awareness pós-aulas)
- Bloco de prova antes da oferta: depoimentos coletados durante aulas
- Oferta: stack completo com âncora de valor
- Urgência: razão real para agir agora

**Post de Grupo — Abertura:**
```
🔓 Aberto.

[Produto] está disponível a partir de agora.

Quem participou das aulas já sabe o porquê.

Para quem é novo aqui: [1 linha de promessa do produto]

[Link] — disponível até [data]
```

## FASE 5 — Pós-Lançamento (D+1 após fechamento)

### Reativação de Não-Compradores (48h após fechamento)
Usar somente se tiver algo real para oferecer:
```
[Nome],

O [produto] fechou ontem.

Mas recebi mensagens de pessoas que queriam entrar e não conseguiram a tempo.

Se você é uma dessas pessoas — e tem [objeção específica que pode resolver] — me responde aqui.

Tenho [X vagas / condição especial] para quem me responder até [data].
```

### Upsell Imediato (para quem comprou — D0 imediato)
Usar template de [OTO em vsl-sales-page-templates.md].
**Tom:** Não interrompa o estado de compra. Celebre a decisão. Apresente a lacuna que o upsell preenche.

### Downsell (para quem recusou o upsell)
```
Tudo bem, entendo que [produto do upsell] pode não fazer sentido agora.

Uma opção: [versão mais simples / menor investimento / só uma parte].

Por R$[X menor], você leva [o mais valioso do upsell].

[Link] → disponível por [24h]
```

**Pré-lançamento:**
- [ ] Copy de CPL criada e aprovada antes de subir os ads?

- [ ] Posts de feed programados (3 antes do CPL + 3 de aquecimento)?
- [ ] Script das 3 aulas escrito e revisado?
- [ ] Broadcasts WhatsApp redigidos para cada fase?

**Carrinho:**
- [ ] VSL de abertura pronto (se aplicável)?
- [ ] Sequência de emails (D0 manhã, D0 noite, D+3, D+5, DC-2, DC-1, DC)?
- [ ] Broadcasts WhatsApp de carrinho redigidos?
- [ ] OTO / Upsell com copy própria?
- [ ] Downsell configurado?

**Pós-lançamento:**
- [ ] Email de reativação (somente com oferta real)?
- [ ] Sequência de onboarding para quem comprou?
