# Pre-Write Brief — Gate Obrigatório Antes de Escrever

> "Se você não consegue preencher este documento, não está pronto para escrever. Copy sem brief é invenção — não persuasão."

O Pre-Write Brief é uma página de checagem intelectual. Força o copywriter a ter respostas claras antes de escrever a primeira linha. Um brief bem preenchido = copy mais rápida, mais precisa e com maior probabilidade de converter na primeira versão.

**Tempo mínimo para preencher:** 30-60 minutos (após research)
**Dependência:** Precisa da VOC coletada (ver [voc-methodology.md](voc-methodology.md))

## BLOCO 2 — DIAGNÓSTICO DE AWARENESS

```
Awareness Level do prospect (1-5):
  [ ] 1 — Unaware: não sabe que tem o problema
  [ ] 2 — Problem Aware: sabe que tem o problema, não conhece soluções
  [ ] 3 — Solution Aware: conhece soluções, não conhece este produto
  [ ] 4 — Product Aware: conhece este produto, não comprou ainda
  [ ] 5 — Most Aware: conhece, comprou ou está pronto para comprar

Sofisticação do mercado (Fase 1-5):
  [ ] Fase 1 — Promessa direta funciona ("Emagreça 10kg")
  [ ] Fase 2 — Promessa com amplificação ("Emagreça 10kg em 30 dias")
  [ ] Fase 3 — Precisa de mecanismo único ("Emagreça com o Protocolo X")
  [ ] Fase 4 — Precisa de novo inimigo ou nova crença
  [ ] Fase 5 — Mercado saturado: anti-promessa ou sub-nicho

Justificativa do diagnóstico (o que na VOC me fez escolher este nível?):
```

## BLOCO 3 — DRE — DIFICULDADE/RAIVA/EMOÇÃO DOMINANTE RESIDENTE

```
DRE primária:
  [ ] Medo (de algo ruim que pode acontecer)
  [ ] Vergonha (de algo que já aconteceu ou que é verdade sobre si mesmo)
  [ ] Raiva (de alguém ou algo que causou o problema)
  [ ] Frustração (tentativas que não funcionaram)
  [ ] Desejo (algo que quer desesperadamente)
  [ ] Culpa (responsabilidade pelo fracasso)

Nível de Escalada Emocional das quotes coletadas (1-5):
  [ ] Nível 1 — Físico ("estou acima do peso")
  [ ] Nível 2 — Consequências ("minhas roupas não fecham")
  [ ] Nível 3 — Relacionamentos ("meu marido não me olha mais da mesma forma")
  [ ] Nível 4 — Identidade/valor próprio ("não me reconheço mais")
  [ ] Nível 5 — Existencial ("já desisti de mim mesma")

Tenho quotes de nível 4-5 na minha VOC? [ ] SIM  [ ] NÃO
(Se NÃO: fazer mais research antes de continuar)
```

## BLOCO 5 — MECANISMO

```
MUP — Mecanismo Único do Problema (Sexy Cause):
[O que realmente causa o problema — não a causa óbvia, mas a causa nova e específica]

Teste do MUP — responder as 3 perguntas:
1. O prospect já ouviu essa explicação antes? [ ] SIM (ruim) [ ] NÃO (bom)
2. Esta causa absolveu o prospect da culpa pelo fracasso? [ ] SIM (bom) [ ] NÃO (ruim)
3. Esta causa cria necessidade única para a nossa solução? [ ] SIM (bom) [ ] NÃO (ruim)

MUS — Mecanismo Único da Solução:
  Gimmick Name (nome próprio do mecanismo): _______________
  Ingrediente Hero (o elemento central): _______________
  Authority Hook (por que existe / quem criou / onde foi descoberto): _______________

Fórmula Ramalho preenchida:
"Já ouviu falar [desse/dessa] [GIMMICK NAME] que [AUTHORITY / GRUPO] estão usando para [RESULTADO DESEJADO]?"

Preenchida: "Já ouviu falar _______________________________________________"
```

## BLOCO 6 — ONE BELIEF (16 Palavras)

Evaldo Albuquerque: a copy inteira tem um único trabalho — fazer o prospect acreditar em uma crença.

```
Fórmula:
"[NOVA OPORTUNIDADE] é a chave para [DESEJO]. Só possível através do [GIMMICK] exclusivo do [PRODUTO]."

Preenchida (máximo 20 palavras):
"_______________________________________________"

Teste: se o prospect acreditar nessa frase, ele vai comprar? [ ] SIM  [ ] NÃO
(Se NÃO: reescrever a frase até que a resposta seja SIM)
```

## CHECKLIST FINAL DE ENTRADA

Só começar a escrever se todos os itens estiverem marcados:

```
[ ] MUP tem Sexy Cause que gera "nunca ouvi isso antes" no prospect?
[ ] MUS preenche a Fórmula Ramalho completa (sem lacunas)?
[ ] VOC tem pelo menos 3 quotes de nível 4-5 da Escalada Emocional?
[ ] One Belief está em ≤ 20 palavras e, se crida, garante a compra?
[ ] Awareness Level foi diagnosticado pela VOC (não chutado)?
[ ] As 3 crenças falsas têm forma de destruição (história ou dado)?
[ ] O tipo de lead bate com o awareness level diagnosticado?
[ ] A oferta tem âncora de preço antes da revelação do preço?
[ ] A urgência/escassez é genuína e verificável?
[ ] A garantia está posicionada como argumento (não como nota de rodapé)?
```

**Se algum item está sem resposta: pare. Volte para a research antes de escrever.**
